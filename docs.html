Welcome to OpenTCAD Documentation

OpenTCAD is a cloud-based semiconductor process emulation platform. It runs a voxel-based emulation engine on the server and delivers interactive 3D results directly in your browser.

This documentation covers how to use the emulator, understand process recipes, and integrate with the API.

What You Can Do

  • Run semiconductor process emulations from your browser
  • Visualize 3D device structures at each process step
  • Explore how parameter changes affect the final structure
  • Export results in glTF format for further analysis

  -H "Content-Type: application/json" \
  -d '{"recipe": "finfet_default"}'
